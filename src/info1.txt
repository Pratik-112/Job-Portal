## About Transpiling ->
## What is Transpiling?
# Transpiling is the process of converting source code from one version of a programming language to another while keeping the same functionality. In the case of React and JavaScript, transpiling is commonly used to convert modern ES6+ (ECMAScript 2015 and later) code into older ES5 code, which is supported by most browsers.

## Why is Transpiling Needed?
# Not all browsers support the latest JavaScript features (like optional chaining, async/await, JSX, etc.). Transpiling ensures that your code runs smoothly in all environments.

## Transpilers in React ->
# Babel (Commonly used, but slower)
# SWC (Speedy Web Compiler) (Faster alternative to Babel, written in Rust). SWC (Speedy Web Compiler) is a fast Rust-based compiler that replaces Babel for transpiling JavaScript/TypeScript code.
# esbuild (Another fast transpiler)
